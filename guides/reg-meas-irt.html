Understanding the 2PL Model
理解 2PL 模型
Item Response Theory (IRT) models the probability that a person with ability θ correctly answers an item. The 2-parameter logistic (2PL) model is:
项目反应理论 (IRT) 对能力为 θ 的被试答对项目的概率进行建模。2参数逻辑模型 (2PL) 为：
P(correct | θ, a, b) = 1 / (1 + exp(-a(θ - b)))

• θ: person's latent ability (x-axis, ranges -4 to +4)
• b: difficulty parameter (location; shifts curve left/right; where P=0.5)
• a: discrimination parameter (slope; steepness of curve; reflects item informativeness)
• P(correct): probability of correct response (y-axis, 0 to 1) • θ：被试的潜在能力（x轴，范围 -4 到 +4）
• b：难度参数（位置；左右移动曲线；P=0.5的位置）
• a：区分度参数（斜率；曲线的陡峭程度；反映项目信息量）
• P(correct)：答对的概率（y轴，0 到 1）
Rasch Model (1PL): The Rasch model is the special case where a=1 for all items. All items discriminate equally; only difficulty differs. This is more restrictive but often more parsimonious.
Rasch 模型 (1PL)：Rasch 模型是所有项目的 a=1 的特殊情况。所有项目的区分度相同，只有难度不同。这更具限制性，但通常更简洁。

What the Simulation Teaches
模拟器的教学要点
Higher b: Curve shifts right → harder item. A person needs higher ability to have a 50% chance of success.

Higher a: Steeper curve → more discriminating. The item strongly differentiates high-ability from low-ability people.

Low-a items: All ability levels have moderate probability → not very informative. Even very low-ability people have a decent chance.

Item information: I(θ) = a² · P(θ) · (1-P(θ)). Information is maximized at θ = b (the difficulty point), where the curve is steepest and uncertainty is highest.

Rasch model: All items have the same shape (a=1), just shifted left/right. This is more restrictive but ensures all items are equally informative across the ability range.
更高的 b：曲线向右移动 → 项目更难。需要更高的能力才能有 50% 的成功概率。

更高的 a：曲线更陡峭 → 区分度更高。项目强烈区分高能力和低能力的人。

低 a 项目：所有能力水平都有中等概率 → 信息量不大。即使低能力的人也有合理的机会。

项目信息：I(θ) = a² · P(θ) · (1-P(θ))。信息在 θ = b（难度点）处最大化，曲线最陡峭，不确定性最高。

Rasch 模型：所有项目具有相同的形状 (a=1)，只是左右移动。这更具限制性，但确保所有项目在整个能力范围内信息量相等。
R Code: Fitting IRT Models
R 代码：拟合 IRT 模型
Load libraries and prepare data
加载库和准备数据
library(ltm)
library(mirt)

# items_matrix: rows=respondents, cols=items (0/1 coding)
# Example: 500 respondents, 10 items (binary responses)
Fit Rasch model (1PL)
拟合 Rasch 模型 (1PL)
rasch_model <- rasch(items_matrix)
summary(rasch_model)
plot(rasch_model, type = "ICC")  # Item characteristic curves
Fit 2PL model
拟合 2PL 模型
twopl_model <- ltm(items_matrix ~ z1)
summary(twopl_model)
plot(twopl_model, type = "ICC")

# Extract item parameters (a and b)
coef(twopl_model)
Estimate person abilities (factor scores)
估计被试能力 (因子得分)
factor.scores(twopl_model)  # Person ability estimates (θ)
Model comparison: Rasch vs 2PL
模型比较：Rasch vs 2PL
anova(rasch_model, twopl_model)
# Likelihood ratio test: is 2PL significantly better than Rasch?
Item fit statistics
项目拟合统计
item.fit(rasch_model)  # Chi-squared tests for each item
# p > 0.05: item fits the model well
