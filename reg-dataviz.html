Data Visualization

Statistics · Foundations 02
What Is This?
这一页讲什么？

A table of numbers tells you almost nothing. A well-designed chart tells you everything — fast. This page covers the principles of effective data visualization and how to produce publication-quality graphics in R.

一张数字表格几乎什么都说明不了。一张设计好的图表能快速说明一切。本页介绍有效数据可视化的原则，以及如何在 R 中制作发表级图表。

Prerequisites
前置知识

Basic familiarity with data and R helpful but not required.

对数据和 R 有基本了解会有帮助，但非必须。

Software & Tools
软件工具

R (ggplot2). A tiny bit of JavaScript for interactive charts.

R（ggplot2）。少量 JavaScript 用于交互图表。
